here's some clarification on tests

Cypress tests are the tests that simulate a user going through the app in a browser and clicking differnt things

Circle tests are unit tests that are purely command line, they just test the funcitonality and logic of the code

to run unit tests locally:

get into the docker api container
    `docker-api`

to run all the tests in a given file, in the main app folder run this command:
    `./vendor/bin/phpunit --filter=<the class name of the file containing the tests you wish to run>`
    Example: `./vendor/bin/phpunit --filter=PPPDealAutoStartGetAffiliateLoanProductTest`

to run just a single test in a given file, in the main app folder run this command:
    `./vendor/bin/phpunit --filter <method name of the test you want to run> <full path to the file containing the test>`
    Example: `./vendor/bin/phpunit --filter testNoPPPLoanProduct tests/Unit/Jobs/PPPDealAutoStart/PPPDealAutoStartGetAffiliateLoanProductTest.php`
    Example: `./vendor/bin/phpunit --filter testIdentifyNameMismatchCorrectly tests/Unit/Libs/ApplicationRequirementService/ApplicationRequirementServiceTest.php`

here's an example of a query to display something grouped by week without having to union each week separately, it generates the week record dynamically

SELECT WEEK(`created`) AS `weekNum` 
    , DATE_SUB(DATE_ADD(MAKEDATE(2021, 1), INTERVAL WEEK(`created`) WEEK), INTERVAL WEEKDAY(DATE_ADD(MAKEDATE(2021, 1), INTERVAL WEEK(`created`) WEEK)) DAY) AS `weekStartDate`
    , SUM(IF(`type` = 'call',1,0)) AS `callCount`
    , SUM(IF(`type` = 'email',1,0)) AS `emailCount`
    , SUM(IF(`type` = 'text',1,0)) AS `textCount`
    , COUNT(DISTINCT `id`) AS `totalCount`
FROM (
        SELECT DISTINCT 'call' AS `type`
            , `c`.`id`
            , `c`.`created`
        FROM `deals` AS `d`
        LEFT JOIN `calls` AS `c` ON `c`.`borrowerId` = `d`.`borrowerId`
            AND `c`.`deleted` IS NULL
            AND `c`.`created` >= '2021-01-11 00:00:00'
        WHERE 1=1
            AND `d`.`deleted` IS NULL
            AND `d`.`created` >= '2021-01-11 00:00:00'
            AND `d`.`loanProductCategoryId` IN (17, 18)
        UNION
        SELECT DISTINCT 'email' AS `type`
            , `m`.`id`
            , `m`.`created`
        FROM `deals` AS `d`
        LEFT JOIN `messages` AS `m` ON `m`.`borrowerId` = `d`.`borrowerId`
            AND `m`.`deleted` IS NULL
            AND `m`.`created` >= '2021-01-11 00:00:00'
        WHERE 1=1
            AND `d`.`deleted` IS NULL
            AND `d`.`created` >= '2021-01-11 00:00:00'
            AND `d`.`loanProductCategoryId` IN (17, 18)
        UNION 
        SELECT DISTINCT 'text' AS `type`   
            , `s`.`id`
            , `s`.`created`
        FROM `deals` AS `d`
        LEFT JOIN `sms` AS `s` ON `s`.`borrowerId` = `d`.`borrowerId`
            AND `s`.`deleted` IS NULL
            AND `s`.`created` >= '2021-01-11 00:00:00'
        WHERE 1=1
            AND `d`.`deleted` IS NULL
            AND `d`.`created` >= '2021-01-11 00:00:00'
            AND `d`.`loanProductCategoryId` IN (17, 18)
    ) AS `der`
GROUP BY `weekNum`
HAVING `weekNum` IS NOT NULL;
